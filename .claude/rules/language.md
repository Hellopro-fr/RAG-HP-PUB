# Language Rules

- This project's CLAUDE.md files are written in English for maximum instruction adherence.
- ALWAYS respond in the same language the user is writing in.
- If the user writes in French, respond entirely in French (explanations, comments, commit messages).
- If the user writes in English, respond entirely in English.
- Code identifiers (variable names, function names, class names) stay in English regardless of conversation language.
- Code COMMENTS follow the conversation language: French comments for French users, English comments for English users.
- Git commit messages: generate in BOTH English and French (see @.claude/rules/commit-messages.md).
